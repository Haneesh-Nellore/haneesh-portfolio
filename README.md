# Haneesh Nellore — Personal Portfolio

A modern portfolio website showcasing AI-native engineering work.

## Tech Stack
- Next.js 14
- TypeScript
- Tailwind CSS
- Framer Motion
- HTML5 Canvas

## Features
- Scrollytelling hero section
- Glassmorphism design
- Projects showcase
- Contact form
- Dark theme

## Getting Started

```bash
npm install
npm run dev
```

Visit `http://localhost:3000`
